# Design System Reference

This folder contains design system documentation used as reference for UI generation with Claude and other AI agents.

## Files

| File | Description |
|------|-------------|
| `DESIGN.md` | Miro-inspired design system — colors, typography, components, layout principles |

## Usage

Reference `DESIGN.md` when prompting Claude or any AI agent to generate UI components. The tokens in that file map directly to Tailwind classes and shadcn/ui component props.

**Primary interactive color:** Blue 450 (`#5b76fe`)  
**Display font:** Roobert PRO Medium  
**Body font:** Noto Sans  
**Text:** Near Black (`#1c1c1e`)
